## Contribution

- **Fork**, **Clone** & **start** the repository on your local.
```bash
git clone
```

After cloning,
```bash
cd frontend-resources
```

Install NPM Packages
```bash
npm i
```

Start the Local Development
```bash
npm start
```

- Make changes in the `docs`.
- Follow good **Code of Conduct**.

- Commit, Push & Create Pull Request.
